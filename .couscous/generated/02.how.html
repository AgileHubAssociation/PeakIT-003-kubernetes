How do we do it ?

Build a kubernetes cluster of course :)

  • Build a Team which always uses a git repository
  • Dockerize the application/s
  • Use a Continous integration / delivery / deployment

Layer 1 - Physical

  • Firewall
  • Load balancer machine: nginx / haproxy
  • Kubernetes worker nodes must be an odd number 3, 5, 7.. so that a leader could be elected
  • Orchestrator software
  • File system / Databases / Mail servers

Some chosen Hardware

  • CPUs: SSe4.2 minimum, nginx needs it now, speed is king for optimizations Intel Xeon E3 1240 v2 - 45~70$

  • Motherboard: Ipmi, Ecc, Low power gpu, Rack friendly, Cheap Intel S1200BTL - 30~60$

  • Memory: Udimm 1Rx8, Ecc and low latency, Databases: also the more the better Kingston KVR1333D3E9S/8G - 40~60$

  • PSU: active pfc, Ups: pure sine wave SilentiumPC Elementum E2, 80+, 450W Eaton 5115

  • SSD: MLCC for long life
